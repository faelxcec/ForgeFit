# ForgeFit
> Backend fitness platform focused on Java, Spring Boot and REST API development.

ForgeFit é um projeto backend desenvolvido com Java e Spring Boot com foco em estudos práticos de desenvolvimento backend, autenticação, persistência de dados e arquitetura de APIs REST.

O objetivo do projeto é evoluir gradualmente para uma plataforma de gerenciamento de treinos, permitindo autenticação de usuários, criação de treinos personalizados, acompanhamento de evolução e outras funcionalidades voltadas para fitness e performance.

## 🚀 Objetivos do Projeto

* Aprofundar conhecimentos em Java e Spring Boot
* Praticar desenvolvimento de APIs REST
* Trabalhar com autenticação e segurança
* Aprender persistência de dados com PostgreSQL e JPA/Hibernate
* Aplicar boas práticas de arquitetura backend
* Evoluir conhecimentos em versionamento com Git/GitHub
* Futuramente integrar cloud e deploy com AWS
